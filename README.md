
  Программа для моделирования формирования устойчивых
   логистических альянсов транспортных компаний

1. НАЗНАЧЕНИЕ ПРОГРАММЫ

Программа предназначена для моделирования кооперации транспортных 
компаний при организации грузоперевозок. Позволяет:

- Генерировать транспортную сеть и компании с клиентами.
- Строить характеристическую функцию кооперативной игры.
- Рассчитывать распределения затрат тремя методами: пропорциональное, вектор Шепли, n-ядро.
- Анализировать устойчивость полученных распределений.
- Проводить серии экспериментов с варьированием параметров.
- Сохранять и загружать параметры и транспортную сеть.
- Экспортировать результаты серии экспериментов в CSV.

2. ЗАПУСК ПРОГРАММЫ

Запустите файл LogisticAlliance.exe.

3. КРАТКАЯ ИНСТРУКЦИЯ

1) Установите параметры сценария в верхней части окна.
2) Отметьте нужные методы распределения (флажки).
3) Нажмите кнопку "Запустить расчёт".
4) Результаты отобразятся на вкладках:
   - Сценарий — транспортная сеть, компании и клиенты.
   - Характеристическая функция — затраты всех коалиций.
   - Распределения — доли затрат по каждому методу.
   - Устойчивость — анализ принадлежности C-ядру.
   - Графики — график эксцессов коалиций.

5) Для серии экспериментов нажмите "Серия экспериментов",
   выберите варьируемый параметр, задайте диапазон и шаг,
   нажмите "Запустить серию". Результаты можно сохранить в CSV.

4. СИСТЕМНЫЕ ТРЕБОВАНИЯ

- Операционная система: Windows 10 / 11 (64-bit).
- Оперативная память: не менее 4 ГБ.
- Свободное место на диске: не менее 500 МБ.
- Установка дополнительных программ не требуется.

5. ОГРАНИЧЕНИЯ

- Число компаний: от 2 до 6.
- Число клиентов: от 2 до 40.
- Число узлов сети: от 5 до 50.
